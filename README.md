# Sinapsis

SINAPSIS (Sistema Inteligente de Análisis de Patrones de Salud Integrados) es una plataforma de historia clínica electrónica unificada para el sector salud colombiano.

- **Frontend**: Next.js 16 + React 19 + Tailwind v4 (importado desde Figma)
- **Backend**: Go (por implementar)
- **Base de datos**: PostgreSQL 16 en Docker
- **IA médica**: modelos pre-entrenados MONAI (análisis de imágenes)
